# Phonebook Backend - Part 3

This repository contains the backend and production build for the Phonebook application developed as part of the Full Stack Open course.

## Live Application
You can access the deployed application in production here:
[Phonebook](https://fullstackopen-part3-backend-5aqa.onrender.com)

## Features
* Express backend serving static production files from React.
* Log management using Morgan middleware.
* Deployed on Render.
